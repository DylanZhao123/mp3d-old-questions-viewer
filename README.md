# MP3D Imagination Question Viewer

Static GitHub Pages viewer for the MP3D A/B/C imagination question set and the generated modality evidence.

- The `Question` tab shows View A, View B, and the question by default.
- View C and the oracle answer remain hidden until `Show hidden C` is enabled.
- The `Modalities` tab shows connectivity graphs, graph distance, 360 heading sweeps, skybox panoramas, depth maps, and room labels.
- The old question JSONL is preserved in `questions_hard_c_only_old.jsonl`.
- Structured web data is in `viewer_data.json`.
- Experiment handoff notes are in `handoff/mp3d_abc_experiment_handoff_20260624.md`.
