# MP3D Old Questions Viewer

Static GitHub Pages viewer for the older MP3D A/B/C hard question set.

- The page shows View A, View B, and the question by default.
- View C and the oracle answer are hidden until the `Show hidden oracle` toggle is enabled.
- The old question JSONL is preserved in `questions_hard_c_only_old.jsonl`.

Open `index.html` locally or use the GitHub Pages URL.
